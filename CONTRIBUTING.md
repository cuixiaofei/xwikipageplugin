# Contributing | 贡献指南

感谢你对本项目的关注与贡献！  
Thank you for your interest in contributing to this project!

---

## How to Contribute | 如何参与

你可以通过以下方式贡献：

- 提交 Issue（Bug / 建议）
- 提交 Pull Request
- 改进代码或文档

You can contribute by:

- Reporting issues (bugs or suggestions)
- Submitting pull requests
- Improving code or documentation

---

## Workflow | 基本流程

1. Fork 本仓库  
2. 创建分支  

   ```bash
   git checkout -b feature/your-feature
   ```

3. 完成修改并提交
4. 推送到你的 Fork
5. 创建 Pull Request 并简要说明修改内容

---

## Guidelines | 规范说明

* 保持代码风格一致

* 一个 PR 尽量只做一件事

* 提交信息清晰简洁

* Keep code style consistent

* One PR for one purpose

* Write clear commit messages

---

## Issues & PRs | Issue 与 PR

* 提交 Issue 前请先搜索是否已存在

* PR 请说明修改原因和内容

* Search existing issues before opening a new one

* Describe **what** and **why** in your PR

---

## Code of Conduct | 行为准则

请保持尊重与友善的沟通环境。
Be respectful and professional in all interactions.

---

Thanks for contributing! ❤️
感谢你的贡献！
